Projects & Research

BuddyBot — Intelligent Delivery Robot

ROS2-based human-following robot on Turtlebot3 using YOLO + MediaPipe for real-time tracking and gesture-driven FSM control. Features visual servoing, auto-recovery, and an adaptive lifting platform.

MuSHR — Full-Stack Autonomous Navigation

Full-stack self-driving system on NVIDIA Jetson Orin + ROS. Implements particle-filter localization (LiDAR + odometry), Halton-sampled roadmap with Lazy A*, Dubins Path planning, and MPC real-time control.

The Devour — Hardware-Integrated Game

CircuitPython survival game on ESP32 integrating accelerometer motion control, rotary encoder input, OLED display, NeoPixel LEDs, buzzer feedback, persistent high-score storage, and a custom 3D-printed enclosure.

Foodie Forum — Microservices Community

Food-sharing community built with Spring Boot microservices (auth, articles, recipes) and a Vue.js frontend. Designed the personal profile UI/UX in Figma and implemented full front-to-back.

Personalized Robot Disambiguation based on User State

Neural Reward Model recalibrating "Ask/Wait/Act" autonomy based on real-time emotional states. RLHF Decision Module via DPO learns optimal interaction strategies from human feedback. (Jan.–Mar. 2026, UW)

Long-tail Bias Mitigation in HOI Detection

Training framework on QPIC Transformer for Human-Object Interaction detection. ATL-inspired parameter enhancement + pseudo-labeling boosted mAP by 1.6% and tail-class accuracy by 5.1%–6.3% on V-COCO. (Dec. 2024–May 2025, Tongji)

Exploring Joyhack in Games Using AI NPCs

User study on how players tamper with AI NPCs using observational methods and semi-structured interviews on the "Suck Up" game. Analyzed player expectations to guide future AI NPC research. (Jan.–Sept. 2024, UC Berkeley)

Algorithm Structure for Real-time General Game Playing Agents

Literature review and proposal of a dual-server (strategy + acting) architecture for real-time general game playing agents, improving breadth and efficiency to better mimic human performance.
